# Dog-Go

We're your dog's other best friend. We've created an app that allows customers to make appointments from local dog walkers. Check it out deployed on heroku. https://blooming-spire-98049.herokuapp.com/

Node
Express
Handlebars
mySql
Sequelize

Moment
```
